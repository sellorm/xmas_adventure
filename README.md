## Objective

The Christmas tree baubles have been stolen and spread around the kingdom.
Find all ten and return them to the tree before the time runs out!

## Controls

D-pad or cursor keys to move.

## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/sellorm/xmas_adventure** and import

## Edit this project ![Build status badge](https://github.com/sellorm/xmas_adventure/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/sellorm/xmas_adventure** and click import
